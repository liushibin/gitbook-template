# book.json 设置

## 基本

* 设置 title、description、auther、language
* 指定 gitbook 版本为 3.2.3
* 添加 sidebar 链接

```
{
  "title": "Gitbook Template",
  "description": "A template for gitbook",
  "author": "Stuart Hua",
  "language": "zh-hans",
  "gitbook": "3.2.3",
  "links": {
    "sidebar": {
        "Blog": "https://blog.stuarthua.com/"
    }
  }
}
```
