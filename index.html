Gitbook Template

基于 Legacy 版 Gitbook 的模版

GitBook 版本为 3.2.3

也推荐 docsify, 相当于一个精简的 GitBook，不需要编译，使用很方便

预览地址：https://stuarthua.github.io/gitbook-template/

使用

  • 安装 node (略)

  • 安装 GitBook

npm install -g gitbook-cli
  • Clone 代码到本地并运行
git clone git@github.com:stuarthua/gitbook-template.git
cd gitbook-template
gitbook install
gitbook serve
  • 在浏览器中查看 http://localhost:4000
