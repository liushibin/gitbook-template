# Gitbook Template

基于 Legacy 版 Gitbook 的模版

## 环境

* node

## 使用

安装 `gitbook-cli`

```bash
$ npm install -g gitbook-cli
```

安装模版所需的插件

```bash
cd gitbook-template
gitbook install
```

运行，在浏览器中查看 [http://localhost:4000](http://localhost:4000)

```bash
$ gitbook serve
```
